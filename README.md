This is a small AI/ML project that predicts students who are at risk academically based on their past performance and background information.
The system helps teachers and institutions to identify students who may need extra support and take preventive actions. 
It uses a Random forest classification and provides a confidence score for each prediction.
Features: 
Predicts if a student will pass or be at risk academically.
Uses student features like:
Gender
Race/Ethnicity
Parental Level of Education
Lunch type
Test Preparation Course
Math, Reading, and Writing Scores
Simple command-line interface for inputting student data.
Outputs prediction and confidence score
